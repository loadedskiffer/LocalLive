# LocalLive

# Installation Instructions

1. Make sure you have Git installed and clone our repository to any location on your machine
2. Install the latest version of Node.js on your machine
3. You must open 2 powershell terminals, one in LocalLive/client and one in LocalLive/server
4. In the client terminal run the command: npm install --force
5. In the server terminal run the command: npm install
6. This will install all of our dependencies
7. To run the app, in both terminals run the command: npm start
8. A browser window will open with the app running
